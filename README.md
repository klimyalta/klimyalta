## Что я умею
- Python — пишу чистый код с обработкой ошибок, многопоточностью и логами.  
- API-интеграции — работаю с биржами через ccxt (Bybit), Telegram Bot API, локальными API (FastAPI/Flask).  
- Машинное обучение (ML) и ИИ (AI) — строю модели: ансамбли RandomForest + XGBoost + LightGBM для предсказания сигналов в трейдинге, обработка данных с pandas/numpy.  
- Frontend — HTML5, CSS (flex/grid, градиенты, анимации, responsive дизайн), JavaScript (vanilla для модалок, чартов, WebSocket).  
- Визуализация — Chart.js для графиков (линейные, stacked, donut), чистый CSS для диаграмм (pie charts).  
- Реал-тайм — WebSocket для live-обновлений, polling как fallback.  
- QA и автоматизация тестирования — пишу тесты на Pytest (unit/integration), автоматизирую проверки API (Requests/ccxt), фронта (Selenium/JS), проверяю ошибки и стабильность.  
- Хранение данных — JSON/CSV/localStorage/файловая система, базово PostgreSQL.  
- Другое — Git, Telegram для уведомлений, PWA (прогрессивные веб-приложения), Tkinter для десктоп-интерфейсов.

### 3. Smart Spot Bot — Веб-дашборд для трейдинга в реальном времени
